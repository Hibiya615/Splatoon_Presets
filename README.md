# Splatoon_Presets
使用Splatoon绘图插件进行的个人绘制存档，目前主要用于CN版。

您可以点击此超链接加入我的Discord仓库以获取更多预设：[点击加入](https://discord.gg/cxaSgg79zu )

并非完全原创，6.x及以后的高难机制大多有前人参考，仅做翻译与部分调整
